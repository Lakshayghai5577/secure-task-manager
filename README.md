# Secure Task Manager API

## Features
- User Registration & Login
- JWT Authentication
- Protected Task Routes
- Password Hashing with bcrypt
- MongoDB Integration

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT

## Setup
1. npm install
2. create .env file
3. npm run dev

## Authentication Flow
- User logs in
- JWT token generated
- Token required for task routes
